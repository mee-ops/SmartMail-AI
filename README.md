
# SmartMail AI 📧🤖

AI-Powered Email Automation built with n8n and Large Language Models (LLMs).

SmartMail AI automatically reads incoming emails, understands customer intent, generates professional draft replies, and saves them directly into Gmail Drafts for human review.

---

## 🚀 Features

- AI-powered email understanding
- Intent detection
- Sentiment analysis
- Automatic draft generation
- Gmail integration
- Human-in-the-loop approval
- Prompt engineering support
- Workflow automation using n8n
- Multi-LLM support
  - OpenAI GPT
  - Claude
  - Gemini
  - DeepSeek

---

## 📌 Workflow

### Step 1

Incoming email received.

### Step 2

Gmail Trigger activates workflow.

### Step 3

Email content is extracted.

### Step 4

AI Agent analyzes:

- Intent
- Sentiment
- Priority
- Context

### Step 5

LLM generates response draft.

### Step 6

Draft saved in Gmail.

### Step 7

Human reviews draft.

### Step 8

Final email sent.
